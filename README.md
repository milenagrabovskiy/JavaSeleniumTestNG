# Java-Selenium-TestNG
This project is for learning and practicing UI test automation using Java and Selenium WebDriver.

## What this project is about
- Practicing Selenium WebDriver for browser automation
- Writing automated UI tests in Java
- Working with Maven project structure
- Using Page Object Model (POM)
- Running tests with JUnit/TestNG

## Tools used
- Java
- Selenium WebDriver
- Maven
- JUnit / TestNG
- IntelliJ IDEA

## How to run tests
You can run tests directly in IntelliJ or with Maven:
mvn test
